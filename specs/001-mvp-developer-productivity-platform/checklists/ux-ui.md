# UX/UI Checklist: MVP Developer Productivity Platform

**Created**: 2026-07-09

## Experience Quality

- [ ] CHK001 The landing page shows a clear hero section, product value, and primary sign-up or sign-in action.
- [ ] CHK002 Authentication, onboarding, and workspace screens use the same visual language and navigation patterns.
- [ ] CHK003 The dashboard allows users to reach core capabilities in at most two clicks.
- [ ] CHK004 Empty states, loading states, and error states are defined for the main screens.
- [ ] CHK005 Interactive controls have visible focus states and consistent spacing.
- [ ] CHK006 Theme switching does not break layout or readability.
