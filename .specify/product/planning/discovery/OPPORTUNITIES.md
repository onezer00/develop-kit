# Oportunidades Estratégicas

Após a pesquisa de mercado, foram identificadas cinco grandes oportunidades:

1. Eliminar a troca de contexto do desenvolvedor.
2. Tornar a busca por utilidades praticamente instantânea.
3. Criar um ambiente consistente, substituindo dezenas de sites isolados.
4. Construir uma plataforma modular preparada para IA e novas capabilities.
5. Diferenciar-se pela experiência do usuário e velocidade, e não pela quantidade de ferramentas.

---

# Critérios para entrar no MVP

Uma oportunidade só poderá ser implementada se atender pelo menos três critérios:

- Resolve uma dor frequente.
- Reduz tempo gasto pelo usuário.
- Possui baixo ou médio esforço de implementação.
- Contribui para retenção.
- É compatível com a visão do produto.

Caso contrário, permanece no backlog de oportunidades.
